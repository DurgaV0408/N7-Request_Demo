# N7-Request_Demo

This repository contains a small React/Vite demo implementing a responsive Hero section inspired by the provided design.

Features:
- Responsive Hero component with typographic scale and spacing similar to the design
- Floating info cards and media mock
- Subtle entrance and hover animations
- Accessibility-focused attributes (aria-labels, focus outlines)

Getting started

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

The app will start with Vite and be available at http://localhost:5173 by default.

Notes
- The image at `src/assets/hero-photo.jpg` is a placeholder; replace it with a higher-res asset if available.
- This demo uses plain CSS in `src/styles.css`; it is structured for easy extraction into CSS modules or a design system.
